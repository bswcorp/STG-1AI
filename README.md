## 🌍 Overview
STG‑AI adalah inti kecerdasan berdaulat dari ekosistem **STG‑Chain**.  
Fungsi utamanya adalah mengubah data mentah, catatan, dan workflow menjadi artefak yang terverifikasi secara diplomatik dan teknologis.

---

## 🧩 Core Components
- **AI Engine** → memproses data mentah menjadi insight terstruktur.  
- **Automation Core** → mengorkestrasi workflow lintas domain (ENERGY, WATER, INFRA, MOBILITY, HUMAN, PUBLIC).  
- **Integration Layer** → menghubungkan STG‑AI dengan QSTATE, STG‑Chain, dan sistem diplomatik global.
